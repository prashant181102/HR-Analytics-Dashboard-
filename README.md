HR Anakytics Dashboard
This project is an interactive HR Analytics Dashboard built using Power BI to analyze employee data and understand workforce trends. 
The dashboard helps track important HR metrics such as employee attrition, department-wise employee distribution, gender analysis, 
average salary, employee satisfaction, and job roles.
The dashboard enables HR teams and management to identify patterns behind employee attrition and make data-driven decisions to 
improve employee retention and performance.

Features :
Employee Attrition Analysis
Department-wise Employee Distribution
Attrition by Gender and Job Role
KPI Cards for HR Metrics
Interactive Filters/Slicers
Data Visualization using Charts & Graphs

Tech Stack Used :
Power BI – Dashboard creation and data visualization
SQL – Data querying and analysis
Excel / CSV – Data cleaning and preprocessing
DAX (Data Analysis Expressions) – Calculated measures and KPIs

Key KPIs:
Total Employees
Attrition Count
Attrition Rate
Average Salary
Average Age
Average Job Satisfaction
Average Years at Company

Insights Generated:
Research & Development department has the highest employee count.
Sales department shows significant attrition.
Male employees have slightly higher attrition compared to females.
Certain job roles contribute more to employee turnover.

Purpose of the Project
The goal of this project is to improve HR decision-making by transforming raw employee data into meaningful insights
through interactive visualizations
